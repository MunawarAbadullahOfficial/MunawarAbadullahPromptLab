## 1. The File Naming Convention
Every file must follow this specific formula:

```
[Category]-[Subject]-[SpecificAction]-[Version].md
```

**Rule:** Use all lowercase and hyphens (no spaces or underscores).

**Why:** Hyphens are better for SEO and web compatibility.

**Examples:**
- `vid-marketing-script-v1.md`
- `pres-investor-deck-v1.md`
- `info-data-visualization-v1.md`
- `gen-history-summary-v1.md`
